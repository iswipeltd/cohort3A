# HR Management System

A role-based HR Management System developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript.

## Features

* Admin Dashboard
* HR Dashboard
* Employee Dashboard
* Employee Management
* Payroll Management
* Attendance Tracking
* Leave Request System
* Authentication & Authorization
* Profile Image Upload
* PDF Payslip Generation
* Search Functionality

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap
* JavaScript
* Dompdf

## System Roles

### Admin

* Full system access
* Manage employees
* Manage payroll
* View reports

### HR

* Manage employee records
* Handle attendance and leave

### Employee

* View profile
* Access payroll
* Submit leave requests

## Project Status

Completed and presented as a full-stack web development project.
